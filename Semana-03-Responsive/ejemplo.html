Cómo funciona el responsive

Cambia el ancho del navegador y observa cuál breakpoint se activa.

/* mobile — defecto */ Móvil < 600px

El estilo base. Mobile First: escribimos aquí primero. Una columna, elementos apilados verticalmente.

@media (min-width: 600px) Tablet ≥ 600px

Pantalla más ancha → podemos mostrar 2 columnas. El contenido se distribuye horizontalmente.

@media (min-width: 900px) Desktop ≥ 900px

Pantalla grande → 3 o 4 columnas. Usamos todo el espacio disponible de manera eficiente.

Unidades de tamaño

Redimensiona el navegador y compara cómo escala cada unidad:

font-size: 32px

Título fijo

Siempre 32px. No cambia con la pantalla. Puede ser muy grande en móvil.

font-size: 2rem

Título con rem

Relativo al tamaño base del documento (16px). Más flexible que px.

font-size: clamp(1.2rem, 4vw, 2.2rem)

Título fluido

Crece y mengua con el viewport. Mínimo 1.2rem, máximo 2.2rem.

Unidad Relativa a Cuándo usarla
px Nada (absoluta) Bordes, sombras, valores exactos
rem Tamaño raíz (<html>) Tipografía, espaciados generales
em Tamaño del elemento padre Padding/margin en componentes
% Dimensión del padre Anchos flexibles en layouts
vw / vh Viewport width / height Héroe a pantalla completa, tipografía fluida
clamp() min / vw / max Tipografía y espaciados completamente fluidos
