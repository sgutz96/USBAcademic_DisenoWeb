# Clase 2: CSS Básico

## Objetivo de la clase

En esta clase aprenderemos sobre:

- El uso básico de **CSS** para el diseño de páginas web.
- Cómo aplicar estilos a los elementos HTML, como **fuentes**, **colores**, **márgenes**, y **paddings**.
- La importancia de la **estructura** y **diseño responsivo** utilizando **CSS**.

## Proyecto en clase

**Mini landing page hecha con CSS**

El objetivo es crear una página de aterrizaje simple usando HTML y CSS. La estructura básica incluirá una cabecera (header), una sección principal (hero), una sección de características (features), y un pie de página (footer).

### Pasos a seguir:

1. **Estructurar el HTML**  
   - Comienza creando una página HTML básica con una estructura estándar.
   - Incluye los elementos esenciales como un **header**, una **sección hero** (o principal), una **sección de características** (features), y un **footer**.

2. **Aplicar los estilos básicos con CSS**  
   - Agrega los estilos básicos para cada sección.
   - Usa el **reset CSS** para evitar inconsistencias entre navegadores.
   - Aplica **colores**, **fuentes** y **espaciado** (márgenes y rellenos) para hacer la página más visual.

3. **Añadir efectos y responsividad**  
   - Añade algunos efectos interactivos como **hover** en los botones e iconos.
   - Usa **CSS Grid** o **Flexbox** para hacer que tu página se vea bien en diferentes tamaños de pantalla (responsividad).

## Tarea para la casa

**Realizar un blog con estilizado básico**

### Requisitos:

- [ ] **Crear la estructura HTML**:
  - El blog debe tener una estructura básica que incluya las siguientes secciones:
    - **Header**: Debe contener un título principal para el blog y un menú de navegación con enlaces a diferentes categorías o páginas.
    - **Main**: La sección principal del blog debe incluir:
      - Un título y una fecha de publicación.
      - Un párrafo de texto como contenido del artículo o entrada del blog.
      - Una imagen o ilustración relacionada con el contenido.
    - **Footer**: Información de derechos de autor, redes sociales o un pequeño texto de contacto.

- [ ] **Crear archivo CSS con el estilizado base**:
  - Estiliza el blog utilizando CSS, con un diseño limpio y agradable.
  - Usa colores y márgenes adecuados para separar las secciones.
  - Los textos deben ser legibles, con una fuente clara y un tamaño adecuado.
  - El diseño debe ser adaptable (responsivo), es decir, se debe ajustar bien en pantallas móviles y de escritorio.
  
- [ ] **Uso de clases y reutilización en el CSS**:
  - Crea **clases reutilizables** para los elementos comunes en tu blog (como `button`, `article`, `header`, `footer`).
  - Usa **herencia** o **selección de elementos comunes** para evitar escribir reglas CSS redundantes.
  - Aplica **recursividad** en el CSS si es necesario (por ejemplo, para menús dentro de menús).

- [ ] **Efecto hover**:
  - Agrega **efectos hover** en los siguientes elementos:
    - Los enlaces en el **header** y en las categorías del menú de navegación. Cambia el color o agrega un subrayado cuando se pase el ratón sobre ellos.
    - El **botón** de contacto o cualquier otro botón que utilices, debe tener un efecto visual cuando el cursor pase por encima, como un cambio de color o un efecto de transformación (escala o sombra).
