# Semana-09-ThreeJS

## 📌 Descripción

En esta semana se introduce el uso de **eventos en JavaScript** y la integración de **Three.js** dentro de interfaces web reales.

A diferencia de ejercicios anteriores donde Three.js ocupa toda la pantalla, en esta clase se explorará cómo **incorporar escenas 3D como parte de la experiencia de usuario**, ya sea dentro de contenedores HTML o como fondos dinámicos que reaccionan al scroll y a la interacción del usuario.

---

## 🎯 Objetivo

Comprender cómo integrar escenas 3D dentro de una interfaz web moderna, utilizando eventos de JavaScript para crear experiencias interactivas y dinámicas.

---

## 🧠 Conceptos clave

### Eventos en JavaScript

* `click`
* `scroll`
* `mousemove`
* `resize`

### Fundamentos de Three.js

* Escena (`Scene`)
* Cámara (`Camera`)
* Renderizador (`Renderer`)
* Objetos 3D (`Mesh`)
* Luces (`Light`)

### Integración con el DOM

* Render dentro de un `div`
* Uso de canvas como fondo
* Relación entre scroll y animación
