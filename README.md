# 📚 Proyecto Web — Curso de Desarrollo Frontend

Este repositorio contiene el desarrollo progresivo de un curso de desarrollo web, organizado por semanas. Cada carpeta representa un módulo de aprendizaje con su respectivo archivo `index.html`.

---

## 🚀 Descripción

Este proyecto recopila ejercicios, prácticas y evaluaciones desarrolladas a lo largo de un curso de desarrollo web frontend, cubriendo desde los fundamentos hasta tecnologías más avanzadas.

Cada semana incluye ejemplos prácticos implementados en un archivo `index.html`, permitiendo visualizar directamente los conceptos aprendidos en el navegador.

---

## ▶️ Cómo Ejecutar

1. Clona este repositorio:

   ```bash
   git clone <URL-del-repositorio>
   ```

2. Abre cualquier carpeta de semana.

3. Ejecuta el archivo `index.html` en tu navegador:

   * Doble clic
   * O con Live Server (recomendado en VS Code)

---

## 📌 Notas

* Cada carpeta es independiente.
* No se requiere backend.
* Algunos ejemplos pueden requerir conexión a internet (CDN o APIs).
