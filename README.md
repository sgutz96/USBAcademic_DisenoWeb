🎨 Curso de Diseño Web — De 0 a Web Developer

Bienvenido/a al repositorio oficial del Curso de Diseño Web.
Aquí encontrarás todos los recursos, ejemplos, proyectos y tareas correspondientes a cada clase.
El objetivo del curso es llevarte desde cero hasta la creación de sitios web completos, responsivos, interactivos y hasta con modelos 3D y frameworks modernos como Vue.

🚀 ¿Qué aprenderás?

Fundamentos de HTML, CSS y diseño web.

Git y GitHub para publicar y versionar proyectos.

Diseño responsive con media queries.

Flexbox y CSS Grid.

TailwindCSS y Bootstrap (opcional).

JavaScript desde cero: DOM, eventos, fetch, APIs, almacenamiento local.

Three.js para crear escenas y modelos 3D en la web.

Vue para construir aplicaciones reactivas y SPA.
