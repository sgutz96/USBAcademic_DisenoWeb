🎨 Curso de Diseño Web — De 0 a Web Developer

Bienvenido/a al repositorio oficial del Curso de Diseño Web.
Aquí encontrarás todos los recursos, ejemplos, proyectos y tareas correspondientes a cada clase.
El objetivo del curso es llevarte desde cero hasta la creación de sitios web completos, responsivos, interactivos y hasta con modelos 3D y frameworks modernos como Vue.

🚀 ¿Qué aprenderás?

Fundamentos de HTML, CSS y diseño web.

Git y GitHub para publicar y versionar proyectos.

Diseño responsive con media queries.

Flexbox y CSS Grid.

TailwindCSS y Bootstrap (opcional).

JavaScript desde cero: DOM, eventos, fetch, APIs, almacenamiento local.

Three.js para crear escenas y modelos 3D en la web.

Vue para construir aplicaciones reactivas y SPA.

📂 Estructura del curso

Cada clase tiene:

Tema principal

Proyecto guiado en clase

Tarea para la casa

| **Clase** | **Tema**                                                | **Proyecto a mostrar en clase**                   | **Tarea para la casa**                                         |
| --------- | ------------------------------------------------------- | ------------------------------------------------- | -------------------------------------------------------------- |
| **1**     | Introducción a diseño web, historia, HTML, Git y GitHub | Primera página HTML + publicación en GitHub Pages | Crear su propia página HTML y subirla a GitHub Pages           |

| **4**     | Flexbox + Grid                                          | Galería o dashboard con grid y flex               | Crear una galería responsive usando grid                       |
| **5**     | Tailwind (Bootstrap opcional)                           | Hero + botones + cards en Tailwind                | Construir un hero/banner usando Tailwind                       |

| **7**     | JS: funciones, clases, DOM y eventos                    | Menú hamburguesa funcional o modal dinámico       | Crear un menú interactivo con JS                               |
| **8**     | DOM avanzado + localStorage + fetch básico              | To-do list persistente                            | Crear una to-do list con localStorage                          |
| **9**     | APIs                                                    | Pequeña app consumiendo API pública               | Consumir una API (Pokémon, clima, RM, etc.) y mostrar tarjetas |

| **11**    | Three.js avanzado                                       | Cargar modelo glTF + iluminación                  | Cargar un modelo 3D e interactuar con él                       |
| **12**    | Entrega 2 + ¿Qué es un framework?                       | Presentación del proyecto 3D                      | Subir su demo 3D con Three.js                                  |
| **13**    | Introducción a Vue + primer proyecto                    | Lista reactiva (tareas/notas)                     | Crear un contador + input reactivo en Vue                      |
| **14**    | Vue: componentes, props y eventos                       | Componentes tipo Card reutilizables               | Crear un componente Card con props                             |
| **15**    | Vue Router + consumo de APIs                            | Mini SPA (2–3 vistas)                             | Crear una mini SPA con routing y consumo de API                |
| **16**    | Entrega final                                           | Presentación del proyecto final                   | Entregar proyecto completo del curso                           |
