Formato de texto en línea

El texto en negrita usa la etiqueta <strong> (indica importancia semántica). El texto en cursiva usa <em> (énfasis semántico). También existe negrita visual con <b> y cursiva visual con <i>, pero sin significado semántico. Texto resaltado con <mark>. Texto tachado con <del> (indica contenido eliminado). Texto subrayado/insertado con <ins>. Subíndice: H2O   |   Superíndice: x2

Lista de definiciones — <dl>

HTML
HyperText Markup Language — lenguaje de marcado para la web.
Etiqueta semántica
Etiqueta HTML cuyo nombre describe el significado del contenido (ej: <article>, <nav>, <footer>).
Atributo
Información adicional dentro de una etiqueta de apertura (ej: class="...", id="...").

Recuerda

  • Las etiquetas semánticas mejoran el SEO y la accesibilidad.
  • El <head> es para metadatos; el <body> para contenido visible.
  • Siempre incluir alt en las imágenes.
  • Asociar <label> con su <input> usando for e id.
  • HTML estructura el contenido; CSS le da apariencia.
