1. Flexbox vs CSS Grid

Antes de aprender cada sistema por separado, es importante entender cuándo usar cada uno. No son alternativas excluyentes: se usan juntos en casi todo proyecto profesional.

📦 Flexbox — Una dimensión
/*
  Ideal para:
  - Barras de navegación
  - Listas horizontales o verticales
  - Alinear elementos dentro de un componente
  - Centrar cosas (vertical + horizontal)
  - Componentes pequeños

  Piensa en Flexbox como
  una línea de elementos.
*/

.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
🗺 CSS Grid — Dos dimensiones
/*
  Ideal para:
  - Layout de página completo
  - Galerías de imágenes
  - Dashboards con múltiples áreas
  - Cuando necesitas controlar
    filas Y columnas al mismo tiempo

  Piensa en Grid como
  una tabla o cuadrícula.
*/

.pagina {
  display: grid;
  grid-template-areas:
    "header"
    "main"
    "footer";
}

Regla práctica: ¿Necesitas controlar solo una dirección (horizontal o vertical)? → Flexbox. ¿Necesitas controlar ambas a la vez? → CSS Grid. ¿No estás seguro? En la práctica, Grid para el layout exterior y Flexbox para los componentes interiores.

2. Flexbox

Flexbox se activa con display: flex en el elemento contenedor. A partir de ese momento, sus hijos directos se convierten en ítems flex y pueden alinearse, distribuirse y redimensionarse de manera flexible.

flex — grow, shrink y basis

La propiedad flex es el shorthand de tres propiedades que controlan cómo cada ítem crece, encoge y cuál es su tamaño base.

flex: 1 | flex: 2 | flex: 3 — distribución proporcional

flex: 0 0 120px — tamaño fijo (no crece, no encoge)

/*
  flex: grow shrink basis
  ├── flex-grow:   cuánto puede CRECER (0 = no crece)
  ├── flex-shrink: cuánto puede ENCOGER (0 = no encoge)
  └── flex-basis:  tamaño BASE antes de distribuir espacio libre

  Valores comunes:
*/
.item-igual    { flex: 1; }          /* todos iguales → flex: 1 1 0% */
.item-doble    { flex: 2; }          /* el doble de espacio */
.item-fijo     { flex: 0 0 200px; }  /* siempre 200px, no cambia */
.item-auto     { flex: 1 1 auto; }   /* crece y encoge desde su tamaño natural */

/* Propiedades que van en los ÍTEMS, no en el contenedor */

/* align-self sobrescribe align-items solo para ese ítem */
.item-especial {
  align-self: center;     /* flex-start | flex-end | center | stretch | baseline */
}

/* order cambia el orden visual (default: 0 para todos) */
/* valores negativos van primero, positivos van al final */
.primero-visual { order: -1; }
.ultimo-visual  { order: 10; }
/* Importante: no afecta el orden del DOM ni la accesibilidad */

3. CSS Grid

CSS Grid se activa con display: grid. A diferencia de Flexbox, permite definir una cuadrícula de filas y columnas y colocar los elementos en posiciones exactas dentro de ella.

/*
  Formas de definir columnas:
*/
.grid {
  display: grid;

  /* Tres columnas fijas de 200px */
  grid-template-columns: 200px 200px 200px;

  /* Tres columnas iguales que ocupan el espacio disponible */
  grid-template-columns: repeat(3, 1fr);

  /* Columnas de distinto tamaño */
  grid-template-columns: 200px 1fr 2fr;

  /* Columnas automáticas RESPONSIVE sin media queries */
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  /*
    auto-fill → crea tantas columnas como quepan
    minmax(200px, 1fr) → mínimo 200px, máximo 1 fracción
    Esto crea un grid que se adapta solo al viewport.
  */
}

grid-template-rows — definir filas

grid-template-rows: 60px 100px 40px

/*
  grid-template-rows funciona igual que columns,
  pero para las filas.
*/
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 60px 100px 40px;  /* fila 1, 2 y 3 */

  /* grid-auto-rows: 80px; → todas las filas generadas automáticamente miden 80px */
}

grid-column y grid-row — spanning

Un ítem puede ocupar más de una celda usando span o especificando las líneas de inicio y fin.

Ítems que ocupan varias celdas
col span 2
normal
normal
row span 2
normal
col span 2
span 4 (toda la fila)
/*
  Los ítems pueden ocupar múltiples celdas con span.
  Las líneas del grid se numeran desde 1.
  Una grilla de 4 columnas tiene 5 líneas verticales (1, 2, 3, 4, 5).
*/
.item-ancho    { grid-column: span 2; }      /* ocupa 2 columnas */
.item-alto     { grid-row:    span 2; }      /* ocupa 2 filas */

/* Especificando líneas exactas: inicio / fin */
.item-preciso  { grid-column: 1 / 4; }      /* de la línea 1 a la 4 (3 columnas) */
.item-exacto   { grid-column: 2 / span 3; } /* empieza en 2, ocupa 3 columnas */
.item-final    { grid-column: 3 / -1; }     /* de columna 3 hasta el final (-1) */

grid-template-areas — nombrar zonas

Esta es la forma más visual y legible de crear layouts. Cada cadena de texto representa una fila, y los nombres definen las áreas.

Layout header / sidebar / main / footer
header
sidebar
main content
footer
/* ── CONTENEDOR: define las áreas y el tamaño de columnas/filas ── */
.layout {
  display: grid;
  grid-template-areas:
    "header  header  header"   /* fila 1: header ocupa toda la fila */
    "sidebar main    main"     /* fila 2: sidebar + main (2 columnas) */
    "footer  footer  footer";  /* fila 3: footer ocupa toda la fila */

  grid-template-columns: 200px 1fr 1fr;   /* ancho de cada columna */
  grid-template-rows: 60px auto 50px;     /* alto de cada fila */
  gap: 1rem;                              /* espacio entre áreas */
}

/* ── ÍTEMS: cada hijo se asigna a su área por nombre ── */
.header  { grid-area: header;  }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main;    }
.footer  { grid-area: footer;  }

/*
  Para hacer el layout responsive, solo hay que cambiar
  grid-template-areas en el media query:
*/
@media (max-width: 600px) {
  .layout {
    grid-template-areas:
      "header"
      "main"
      "sidebar"
      "footer";
    grid-template-columns: 1fr;
  }
}

  ESTRUCTURA DE UN LAYOUT REAL
  Grid para la página → Flexbox para los componentes
*/

/* 1. Grid en el body principal */
.pagina {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  grid-template-columns: 200px 1fr;
  grid-template-rows: 60px 1fr 50px;
  min-height: 100vh;
}

/* 2. Flexbox en el header */
.header {
  grid-area: header;
  display: flex;                   /* ← Flexbox aquí */
  justify-content: space-between;
  align-items: center;
}

/* 3. Grid en las cards internas */
.cards {
  display: grid;                   /* ← Grid aquí */
  grid-template-columns: repeat(auto-fill, minmax(160px, 1fr));
  gap: 1rem;
}

/* 4. Flexbox dentro de cada card */
.card {
  display: flex;                   /* ← Flexbox aquí */
  flex-direction: column;
  gap: 0.5rem;
}

5. Casos de uso comunes

Navbar con Flexbox

El patrón más común: logo a la izquierda, links centrados o a la derecha, botón al extremo.

/* Navbar clásica con Flexbox */
.navbar {
  display: flex;
  justify-content: space-between;  /* logo | links | botón */
  align-items: center;             /* centrado vertical */
  padding: 1rem 2rem;
}

.nav-links {
  display: flex;                   /* links en fila */
  gap: 1.5rem;
  list-style: none;
}

Centrar perfectamente con Flexbox

La técnica más sencilla para centrar un elemento horizontal y verticalmente.

Centrado perfectamente
justify-content: center + align-items: center 
/* La técnica de centrado más limpia en CSS moderno */
.contenedor {
  display: flex;
  justify-content: center;   /* centrado horizontal */
  align-items: center;       /* centrado vertical */
  height: 100vh;             /* necesita altura definida */
}

/* Alternativa con place-items (shorthand) */
.contenedor-v2 {
  display: grid;
  place-items: center;       /* centra en las dos dimensiones */
  height: 100vh;
}

Cheat Sheet — Flexbox y CSS Grid

Propiedad Dónde va Qué hace
display: flex Contenedor Activa Flexbox
flex-direction Contenedor row | column | *-reverse
justify-content Contenedor Eje principal: start/end/center/between/around/evenly
align-items Contenedor Eje cruzado: start/end/center/stretch/baseline
flex-wrap Contenedor nowrap | wrap | wrap-reverse
gap Contenedor Espacio entre ítems
flex Ítem grow shrink basis — cuánto crece/encoge
align-self Ítem Sobrescribe align-items para ese ítem
order Ítem Orden visual (default: 0)
display: grid Contenedor Activa CSS Grid
grid-template-columns Contenedor Define el número y tamaño de columnas
grid-template-rows Contenedor Define el número y tamaño de filas
grid-template-areas Contenedor Nombra zonas del grid visualmente
grid-column / grid-row Ítem Cuántas celdas ocupa el ítem
grid-area Ítem Nombre del área que ocupa

6. Proyecto Aplicado

🗺 Dashboard o Portafolio con Grid + Flexbox

Construye una página con un layout completo que demuestre el uso de Flexbox y CSS Grid de forma combinada. Puede ser un dashboard de administración, un portafolio personal o una tienda con sección de productos.

Requisitos mínimos

  • ✓ Layout de página completo usando grid-template-areas (header, main, footer mínimo).
  • ✓ Navegación horizontal construida con Flexbox y justify-content.
  • ✓ Sección de tarjetas o galería usando grid-template-columns: repeat(auto-fill, minmax(...)).
  • ○ Al menos un elemento con grid-column: span N o grid-row: span N.
  • ○ Tarjetas con botón al fondo usando flex-direction: column y flex: 1.
  • ○ Layout responsive: el grid cambia con un media query (de 3 columnas a 1 en móvil).
  • ○ Uso de gap para los espacios (no margin entre ítems).

Criterios de evaluación

📦 Flexbox 25 pts

  • Navbar con justify-content
  • Uso de align-items
  • Tarjetas con flex-direction: column
  • flex: 1 para distribución

🗺 CSS Grid 25 pts

  • Layout con grid-template-areas
  • Galería con auto-fill / minmax
  • Al menos un span de columnas o filas
  • Uso correcto de gap

📱 Responsive 25 pts

  • Mobile First
  • Grid cambia en breakpoints
  • Navbar colapsable en móvil

✨ Calidad 25 pts

  • Código limpio y comentado
  • Uso de variables CSS
  • Sin uso de float ni position para el layout

💡 Herramientas recomendadas: Flexbox Froggy y Grid Garden son juegos interactivos para practicar. El inspector de Chrome/Firefox tiene herramientas visuales para depurar Grid y Flexbox.

Recuerda: No uses float ni position: absolute para construir el layout. Esas técnicas son de generaciones anteriores de CSS. Flexbox y Grid son la forma correcta y moderna.
